# Curated Results

This directory contains lightweight thesis artifacts only.

Included:

- `tables/`: CSV sources for thesis tables and compact appendix tables.
- `figures/`: selected final figures used in the thesis/report.
- `reliability_diagrams/`: optional curated reliability diagrams.

Not included:

- raw JSON result dumps,
- full training logs,
- checkpoints,
- datasets,
- transfer archives,
- exploratory plots.

Those larger artifacts can be regenerated with the scripts documented in `docs/reproduce_thesis_results.md`.
